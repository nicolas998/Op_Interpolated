CronLogs: Carpeta para guardar los logs de las ejecuciones croneadas del modelo.

Se guardan los siguientes crons:

- Log_ActualizaCampos.txt

Determina si se actualizaron o no los campos de precipitacion segun el radar 
para la cuenca.

- Log_CampoExtrapolated.txt

Determina si se actualizaron o no los campos extrapolados para la cuenca 

- Log_Ejecucion_v01.txt

Determina si se ejecuto o no el modelo de forma correcta


- Log_Figuras_simple.txt

Determina si se generaron o no lass figuras de simulaciones

- Log_Figuras_extrapol.txt

Determina si se generaron o no las figuras de extrapolacion de radar
