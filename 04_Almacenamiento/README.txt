En "Storage" se guardan las condiciones de almacenamiento de cada 
proyecto de simulacion y de cada una de las calibraciones que se tengan 
establecidas. Cada almacenamiento esta acompañado de dos archivos:
.StObin y .Stohdr, el primer archivo contiene los binarios de 
almacenamiento de la cuenca, el segundo contiene el valor 
promedio para toda la cuenca.

Al interior de la carpeta se tiene la carpeta "temp", en esta 
se guardan los estados de almacenamiento transicionales
para la ejecución del modelo.
